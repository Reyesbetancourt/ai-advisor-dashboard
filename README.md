# La IA como Sistema Operativo del Asesor de Negocios

Tablero-portafolio (Streamlit) que muestra **cómo aplico IA** para pasar de la
estrategia a la entrega técnica end-to-end, y **dónde la aplico** en seis ventures reales.

## Tesis
No asesoro desde la teoría: construyo sistemas completos. **Orey AI Assistant** es
mi copiloto transversal, y el plan es replicar un asistente como él dentro de cada
proyecto para convertir cada negocio en un sistema que aprende y se opera con IA.

## Contenido del tablero
1. **El método** — pipeline con IA (6 etapas: research → arquitectura → spec → generación → validación → deploy).
2. **Portafolio** — UEGE, Dentwork, BarberBROTHER, Viajes Zvezda, PatrimonioOS, Orey AI.
3. **Operación en datos** — ventures por vertical, fase de madurez, stack recurrente.
4. **La tesis** — Orey como capa que alimenta cada venture.

## Correr localmente
```bash
pip install -r requirements.txt
streamlit run app.py
```
